# Magisk.it

Create Magisk Modules from Github! (For times when you can't make a zip yourself)

## Live

https://magisk.it (Coming Soon)

## Development

You'll need:

  * [Node.js](https://nodejs.org)
  * [Yarn](https://yarnpkg.org)

```bash
yarn
export DEBUG=magisk.it:*

node index.js

# Check port :8080!
```
